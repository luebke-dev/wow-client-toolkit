## 3. Call-site audit (LEA scaled-index = danger pattern)

For each call site, we score the surrounding 64 bytes of
machine code:

- **+3** if a `lea reg, [reg*N + imm32]` with N>=2 appears in the
  instruction window (scaled-index destination)
- **+2** if the basic block contains a backward conditional jump
  (loop body)
- **+2** if the same function reads from a global (`mov reg, [imm32]`)
  and uses it as the loop bound (packet-controlled iteration count)

Score >= 5 = same shape as the BG vuln. Score 3-4 = worth a manual
look. Score 0-2 = local-variable destination, safe.
